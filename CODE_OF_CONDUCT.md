# Code of conduct

This project follows the [Contributor Covenant](https://www.contributor-covenant.org/version/2/1/code_of_conduct/), version 2.1.

## TL;DR

- Be kind. Assume good faith.
- Disagreement is fine; personal attacks are not.
- Don't post anything you wouldn't want quoted back to you.

## Scope

These standards apply in any space the project occupies — Issues, Discussions, PRs, commits, releases, and any communication directed at the project or its participants.

## Reporting

Conduct concerns can be sent to the project maintainer via email at the address in the `git log` author field, or via a [private vulnerability report](https://github.com/jaenster/puppeteer-mcp-claude/security/advisories/new) (it works for non-security reports too — it's just a private channel). Reports are confidential.

## Enforcement

Maintainer judgement, in line with the Contributor Covenant's [enforcement guidelines](https://www.contributor-covenant.org/version/2/1/code_of_conduct/#enforcement-guidelines). Steps range from a private nudge to removing or banning a participant from project spaces, depending on severity and pattern.
